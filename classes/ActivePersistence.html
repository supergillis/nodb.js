emit

(
  • args
)

Defined in active-persistence.js:82

Available since 1.0.0

Notify the corresponding callbacks that the event has occured. The on callbacks will always be called. The given func function will automatically be executed. When it finishes successfully the corresponding after callbacks will be called.

Parameters:

  • args Object
    • event String

      The name of the event that occurred.

    • [args] Any optional

      The arguments that will be passed to the callbacks.

    • [func] Function optional

      This function will automatically be executed. When it finished successfully the corresponding after callbacks will be called.

    • [caller] Object optional

      The object to be bound to this in the given func.

This method is syntactic sugar for:

all().filter(filter);

Parameters:

  • filter Function

    A function that accepts one parameter, i.e. an instance of the model. Returns true if the iterator must include this instance, false if it must be skipped.

Returns:

Iterator: An iterator for all the instances of this model for which filter returns true.
